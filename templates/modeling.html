模型说明
{% if model_name == 'naive_bayes' %}

高斯朴素贝叶斯是一种基于贝叶斯定理的分类器，假设特征之间相互独立且服从高斯分布。它在分类问题中表现良好，特别是当数据维度较高时。

{% elif model_name == 'neural_network' %}

BP神经网络是一种多层感知器，通过反向传播算法学习模式，能够捕捉数据中的复杂非线性关系。此模型使用ReLU激活函数和Adam优化器。

{% elif model_name == 'logistic_regression' %}

逻辑回归是一种线性分类算法，适用于二分类问题。它通过logistic函数将线性组合转换为概率值，用于预测样本属于某一类别的概率。

{% elif model_name == 'random_forest' %}

随机森林是集成学习方法的一种，通过构建多个决策树并取平均值来进行分类。它能够处理高维数据和大量特征，并评估特征的重要性。

{% elif model_name == 'knn' %}

K最近邻(KNN)算法基于"物以类聚"的原则，通过计算样本与训练集中样本的距离，将其分类为最接近的K个邻居中出现最多的类别。

{% elif model_name == 'bagging' %}

装袋模型(Bagging)是一种集成学习方法，通过对训练数据进行多次随机采样训练多个基分类器，然后进行投票决策。它可以减少过拟合并提高模型稳定性。
